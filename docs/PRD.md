# Star 全屋定制工厂直营官网 — 产品需求文档 (PRD)

## 1. 项目背景

Star 是一家定位于"工厂直营、价格透明、设计领先"的高端全屋定制品牌。本项目为 Star 打造一套覆盖 H5 / 小程序 / PC 三端的品牌官网与运营后台，目标：

1. 在三端呈现一致、克制、奢华的视觉调性。
2. 把工厂直营的全屋定制案例高效地展示给潜在客户，引导留资 / 进店。
3. 提供一个独立的运营后台，支持日常内容运营（Banner、风格标签、热门案例置顶等）。

## 2. 目标用户

| 角色 | 描述 | 主要诉求 |
| --- | --- | --- |
| 业主 (H5/小程序/PC) | 准备装修 / 在选购定制柜的潜在客户 | 浏览风格与案例、估算价格、查看工厂实力 |
| 运营 (PC 后台) | Star 内部运营人员 | 配置首页 Banner、维护风格标签、置顶爆款案例 |
| 管理员 | 内部管理员（默认 13800138000） | 全部运营权限 + 账号管理 |

## 4. 功能需求

### 4.1 前台
- F1 首页 Banner 自动轮播 (3-5 张)，支持跳转内部链接 / 案例详情。
- F2 一级风格标签横向滚动 + 选中态高亮；点击切换下方热门案例数据源。
- F3 二级筛选 (空间 / 颜色 / 尺寸 / 价格) 支持多选；颜色以彩色圆圈呈现。
- F4 热门案例列表支持懒加载瀑布流；置顶案例标记"🔥 爆款"。
- F5 案例详情页支持大图查看、价格展示、规格参数。
- F6 全站响应式：PC ≥ 1280 / 平板 768-1280 / H5 ≤ 768。

### 4.2 账号
- F7 手机号 + 6 位验证码登录 (开发期固定 1234)。
- F8 Token 保存在 LocalStorage，7 天有效期。
- F9 默认管理员账号 13800138000，可进入 `/admin`。

### 4.3 后台
- F10 Banner 增删改查，支持启用 / 禁用。
- F11 一级风格标签维护 (名称、图标、排序)。
- F12 二级标签维护，按"空间 / 颜色 / 尺寸 / 价格"四种维度分组。
- F13 置顶案例管理 (最多 8 个)。
- F14 案例管理：增删改查、富文本详情、图片上传 (本地 URL 占位)。
- F15 操作日志 (写入 MongoDB)。

## 5. 非功能需求

- 性能：首屏 LCP ≤ 2.5s (PC) / 3s (H5)；图片懒加载。
- 兼容：Chrome ≥ 90 / Safari ≥ 14 / 微信内置浏览器。
- 安全：JWT 鉴权、后台接口角色校验、参数校验、防 SQL/XSS。
- 可维护：前端组件化 (React + Vite)；后端分层 (Handler / Service / Repo)。

| AC3 | 每个一级风格至少 5 个真实案例，价格区间真实可信 |
